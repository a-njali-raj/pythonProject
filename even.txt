0.Python is an interpreted, high-level and general-purpose programming language.
2.Its language constructs and object-oriented approach aim to help programmers write clear, logical code for small and large-scale projects.
4.It supports multiple programming paradigms, including structured (particularly, procedural), object-oriented and functional programming.
6.Python was created in the late 1980s, and first released in 1991, by Guido van Rossum as a successor to the ABC programming language.
8.Python 3.0, released in 2008, was a major revision of the language that is not completely backward-compatible and much Python 2 code does not run unmodified on Python 3.
10.Python interpreters are supported for mainstream operating systems and available for a few more (and in the past supported many more).
12.A non-profit organization, the Python Software Foundation, manages and directs resources for Python and CPython development.
